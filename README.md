**Blazingly fast Telegram Group Management Bot based on [Telethon](https://github.com/LonamiWebs/Telethon) coupled with [sqlalchemy](https://github.com/sqlalchemy/sqlalchemy), [mongodb](https://github.com/mongodb/mongo), [falcon](https://github.com/falconry/falcon) and [gunicorn](https://github.com/benoitc/gunicorn).**

Can be found on telegram as [Julia](https://t.me/MissJuliaRobot)
